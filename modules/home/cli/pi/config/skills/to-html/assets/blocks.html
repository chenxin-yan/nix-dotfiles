Auth refactor plan

Replace cookie-based session auth with stateless JWT, in three phases.

2 · Callout (info / warn / risk / good)

Info

Touches every authenticated route; rollback is one feature flag.

Warn

  • Session table stays around for 30 days.
  • Mobile clients ship a separate release.

Risk

Token revocation isn't instant — compromised accounts stay valid until expiry.

Good

Removes Redis from the hot path. Latency drops ~12ms p50.

3 · Compare (2-column)

Before

Server-side session lookup on every request via Redis.

After

Stateless JWT validation in middleware; no Redis hit.
